XML Schema

This is the documentation for the XML Schema modules developed by Andy Wardley at Canon Research Center Europe Ltd.

The overview document gives an overview of XML Schema and describes the purpose and usage of these modules. The modules document describes the different modules that comprise the toolkit and how they fit together and contains links to the detailed technical documentation for each module.
